CALIDAD DEL TAMAÑO DE UN LIENZO DE PINTURA

La profesora de Artes de la Universidad Distrital Francisco José de Caldas dejó como proyecto final a sus alumnos realizar una pintura con los materiales que tengan a la mano, esta debe tener una característica de arte contemporáneo, para rellenar cada lienzo se necesitan 3000 ± 15 miligramos. En este trabajo, se sabe que la mínima cantidad para rellenar el lienzo tiene la misma probabilidad de no terminarse que un lienzo totalmente finalizado. Es decir, la probabilidad de que aleatoriamente se obtenga un lienzo totalmente finalizado es de 3015 miligramos es igual a la probabilidad de obtenerlo con la mínima cantidad de rellenar el lienzo con 2985 miligramos. Se sabe que la probabilidad de que el proyecto cumpla con las especificaciones es de 0.99.


a) ¿Cuál es la probabilidad de que un lienzo elegido aleatoriamente sea muy largo?

b) ¿Cuál es la probabilidad de que un lienzo elegido aleatoriamente sea mayor que 2985 milímetros?

Solución:


Si sabemos que B es el evento, el cual cumple con las especificaciones. S y L los casos que sea muy grande o pequeño, son:

a. P(M) = 0.09 Y P(S) = P(L)=(1-0.09)/2 = 0.005 Con base a esto se puede inferir que 5 de cada 1000 lienzos seran muy largos

Si tenemos que X es el relleno de un lienzo seleccionado aleatoriamente, por lo cual se obtiene:



P(2985 ≤ X ≤ 3015 = P(M) = 0.09

Como P(X ≥2985) = 0.005, entonces

P(X≥2985) = P(M) + P(L) = 0.995

Y utilizando P(A) + P(A ) = 1 se obtiene:

P(X≥2985)+P(X<2985) = 1

Por lo cual, P(X≥2985) = 1 - P(S) = 1-0.005=0.995

Con base a este resultado se puede evidenciar que 995 de cada 1000 lienzos serán más largos de 2986 milímetros.
